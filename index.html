NGU Industries Analyzer & Optimizer

Welcome. You have stumbled upon a NGU Industries file save analyzer and map beacons placement optimizer. Load your save file to view some game progress details. Use the Optimizer section to find near optimal beacons placement on maps, be it for speed, production or a combination of the two.

Warning: There are game spoilers on this page.

This tool is written in GoLang WebAssembly (Wasm). It relies on Go-App and NGUI-Save-Reader . Source code can be found at https://github.com/MisterCodo/ngu .

Select Save File

Load a save file to see game progress details. Furthermore, beacons placement optimization will automatically consider tiles you have yet to unlock in your NGU Industries save file.

Beacons Optimizer

1. Select a map:

2. Select beacon types to place:

3. Select an optimization goal. Speed is useful for labs, production is useful for low speed cap materials and a combination of speed & production is useful otherwise.

4. Click on the map image to block sections you have yet to unlock in the game. Loading a save file handles this automatically.

5. Click Start to optimize the map. The application will block for approximately 15 seconds before displaying results.
